# Mini Bank System (C)

A simple console-based banking system written in C.

## Features
- Create bank accounts
- Deposit money
- Withdraw money
- Display all accounts
- Persistent data storage using files

## Concepts Used
- Structures
- File handling
- Functions
- Menu-driven programming

## How to Run
1. Compile the program:
   gcc main.c -o bank
2. Run:
   ./bank
